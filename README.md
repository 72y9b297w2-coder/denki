## GitHubへの入れ方
既存の `ksei` リポジトリとは別に、`denki` などの新しい公開リポジトリを作る方法が安全です。

1. このZIPをiPhoneの「ファイル」で解凍
2. `index.html`、`style.css`、`app.js`、`questions.js`、`README.md`をアップロード
3. Settings → Pages
4. Source: Deploy from a branch
5. Branch: main / (root)
6. Save
